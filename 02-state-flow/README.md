# Lesson 2: State Flow and Lifecycle

In this lesson, you will practice managing and inspecting the Terraform state file.

## Objectives
1. Understand the role of `terraform.tfstate`.
2. Inspect the state using commands: `terraform state list` and `terraform state show`.
3. Experiment with drift detection (modifying physical assets and running plan).
4. Configure resource lifecycle hooks (e.g. `prevent_destroy`, `create_before_destroy`).
